Awesome SQL

===========

受其他令人敬畏的列表启发的使用关系数据库的工具和技术列表。

  • Oracle-专门用于Oracle和PL/SQL的软件和软件包。
  • SQL-Server-专门用于sql-server和t-sql的软件和软件包。
  • MySQL-已经有完整的MySQL列表。
  • 跨平台的东西
    • 格式化程序-工具和网站，将格式化你丑陋的SQL代码。
    • 工具-其他很酷的工具和网站。

Oracle

  • SQL Developer-用于Oracle数据库的PL/SQL开发和管理的Oracle免费IDE
  • SQLcl-用于Oracle数据库的免费命令行界面。
  • SQL Tools for Oracle-用于Oracle数据库开发的轻量级前端。
  • Oracle Database Express Edition-免费的入门级数据库，用于开发和部署应用程序
  • Alexandria PL/SQL Utility Library-PL/SQL实用程序包的集合
  • PLSQL-JSON-PL/SQL库来编码/解码JSON
  • utPLSQL-PL/SQL的单元测试框架

SQL Server

  • SQL Server Express Edition-免费的SQL Server数据库，用于开发和部署应用程序
  • SQL Server Data Tools-开发人员为MS SQL Server堆栈设计和构建数据库和其他商业智能解决方案的集成环境。
  • tSQLt-SQL Server的单元测试框架
  • MSSqlCLI-SQL Server的命令行客户端，具有自动完成和语法突出显示功能

格式化程序

  • SQL Format-即时SQL格式化程序
  • Poor SQL-即时免费和开源t-sql格式 (在这里查找插件和诸如此类的东西: https://github.com/TaoK/PoorMansTSqlFormatter)
  • T-SQL Tidy-使用Webservice和SSMS插件进行在线t-sql格式化。

工具

  • PixQL-@ Phildo在SQL中的命令行图像处理工具
  • SQL Fiddle-Easly测试和共享数据库问题及其解决方案。可以使用不同的后端DBMS (MySQL、PostgreSQL、MS SQL Server、Oracle和SQLite)
  • SqlPad-用于运行SQL查询和可视化结果的web应用程序
  • ERAlchemy-ERAlchemy从数据库生成实体关系 (ER) 图
  • BigBash-开源转换器，从SQL Select查询生成bash单行，无需数据库
  • Flyway-数据库迁移工具
  • Liquibase-数据库的源代码控制
  • PgCLI-Postgres CLI与自动完成和语法突出显示
  • LINQPad-LINQPad不仅用于LINQ查询，还用于任何C #/F #/VB表达式，语句块或程序。
  • SchemaSpy-我们将尽最大努力简化数据库的文档流程

资源

  • Curated SQL Learning Resources on Hackr.io-编程社区策划的学习SQL的资源
